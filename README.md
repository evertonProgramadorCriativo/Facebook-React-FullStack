 ### Descrição


## Desenvolvi uma Plataforma Social de Notícias com React.js e Context API


### Funcionalidades

 Sistema de Login/Logout completo

 Persistência de sessão usando localStorage

 Rotas protegidas com componente ProtectedRoute

 Context API para gerenciamento de estado global de autenticação

 Design responsivo e visual inspirado no Facebook


![Home Page Imagem](https://i.ibb.co/8Lv9HbV9/Screenshot-50.png)
### Tecnologias

React.js - Biblioteca principal

React Router DOM - Roteamento

Context API - Gerenciamento de estado global

CSS Modules - Estilização

LocalStorage - Persistência de dados
